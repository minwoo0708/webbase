기술 목록

1. JavaScript (자바스크립트)

자바스크립트를 활용하여 동적인 웹 페이지를 구현할 수 있습니다. DOM 조작을 통해 사용자의 이벤트에 실시간으로 반응하는 UI 요소를 제작할 수 있으며, 비동기 통신을 활용하여 외부 API 데이터를 받아와 화면에 렌더링하는 기본적인 프론트엔드 기능을 다룰 줄 압니다.


2. React (리액트)

리액트 프레임워크를 학습하여 컴포넌트 기반의 구조적인 웹 개발이 가능합니다. 상태 관리와 프롭스를 활용하여 재사용성이 높은 UI 컴포넌트를 설계할 수 있으며, 싱글 페이지 애플리케이션 환경에서 유기적이고 빠른 화면 전환 효과를 구현해 본 경험이 있습니다.


3. HTML5 / CSS3

HTML5와 CSS3를 기본으로 웹 표준과 접근성을 준수하는 시멘틱 마크업을 수행합니다. 다양한 레이아웃 구성 기법을 능숙하게 다루며, 키프레임 애니메이션 속성을 적용하여 JavaScript 없이도 부드럽고 생동감 넘치는 시각 효과를 화면의 특정 요소에 부여할 수 있습니다.
